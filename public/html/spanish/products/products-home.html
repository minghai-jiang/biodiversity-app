El producto principal de Ellipsis es una API fácil de usar que le permite consultar y visualizar la gran cantidad de información geográfica de nuestra base de datos. Además, ofrecemos un visor interactivo de código abierto y un servicio de Geomensaje para involucrar fácilmente a la gente en sus proyectos.

API

Ellipsis proporciona una API pública de REST que le permite consultar nuestros datos agregados directamente. También ofrece visualizaciones atractivas que se pueden importar fácilmente en su marco de trabajo favorito, como arcGIS o Leaflet.

Visor

El visor de Ellipsis ofrece un bajo umbral de acceso a todos los datos de Ellipsis. Explore nuestros mapas interactivos y/o datos de exportación para ser procesados en la aplicación de su elección.

GeoMensaje

Geomensaje permite a los usuarios enviar observaciones y comentarios georeferenciados a la base de datos de Ellipsis. Esta retroalimentación se utiliza para corregir modelos e informar a otros usuarios. Geomensaje está integrado tanto en el Visor como en la API pública.

Acerca de los datos

Los datos de Ellipsis se adquieren y procesan con la misma resolución que la fuente de datos original del satélite. Actualmente ofrecemos tres tipos de información:

  • Índices espectrales, para evaluar la condición de una región.
  • Clasificación, para evaluar el tipo de cubierta vegetal de una región.
  • Cambio repentino, para evaluar si ha ocurrido un evento repentino en una región.

Además de los datos generados automáticamente, podemos almacenar mensajes humanos georeferenciados.

Agregamos esta información tanto a los mosaicos webMercator estándar como a los polígonos personalizados que son específicos del mapa.

Accesibilidad

Ellipsis mantiene varios servicios de monitoreo para una amplia gama de clientes. Algunos de estos mapas son públicos y pueden ser vistos y utilizados por cualquier persona, otros son privados y sólo se puede acceder a ellos si se dispone de las credenciales de acceso necesarias.
