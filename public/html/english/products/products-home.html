Ellipsis' main product is an easy to use API that allows you to query and visualize the vast amount of geographic information in our database. On top of this we offer an open-source interactive viewer to easily engage people on your projects.

API

Ellipsis provides a REST API that allows you to query our aggregated data directly. It also offers attractive and interactive visualizations that can be easily imported in your favorite framework, such as arcGIS or Leaflet.

Viewer

The Ellipsis viewer offers low threshold access to all your Ellipsis data. Explore our interactive maps and/or export data to be processed in your application of choice.

GeoMessage

The API and viewer allow user to give georeferenced remarks and feedback in the Ellipsis database. This feedback is used to correct models and inform other users.

About the data

Ellipsis data is acquired and processed on a resolution of 10 by 10 meters. We currently provide three types of data:

  • Spectral indices, to assess the condition of a region.
  • Classification, to assess the land cover type of a region.
  • Sudden change, to assess whether an abrupt event has occurred in a region.

We aggregate data to both standard OpenStreetMap tiles as well as to custom polygons that are map specific.

On top of automatically generated data we can store georeferenced human messages.

Accessibility

Ellipsis maintains various monitoring services for a broad range of customers. Some of these maps are public and can be viewed and used by anyone, others are private and can only be accessed if you have the required login credentials.
