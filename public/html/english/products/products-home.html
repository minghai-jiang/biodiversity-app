Ellipsis' main product is an easy to use API that allows you to query and visualize the vast amount of geographic information in our database. The API includes an elaborate georeferenced chat service. Our customers can built apps and alert services using our API, if needed we can also deliver full service and develop and host these apps and alertservices ourselves within the Ellipsis framework.

API

Ellipsis provides a public REST API that allows you to query our aggregated data directly. It also renders attractive visualizations that can be easily imported in your favorite framework, such as arcGIS or Leaflet.

GeoMessage

Geomessage is an integrated feature in our API that allows users to sent georeferenced remarks and feedback to the the Ellipsis database. This feedback is used to correct models and inform other users.

About the data

Ellipsis data is acquired and processed on the same resolution as the original satellite datasource. We currently provide three types of information:

  • Spectral indices, to assess the condition of a region.
  • Classification, to assess the land cover type of a region.
  • Sudden change, to assess whether an abrupt event has occurred in a region.

On top of automatically generated data we can store georeferenced human messages.

We aggregate this information to both standard webMercator tiles as well as to custom polygons that are map specific.

Accessibility

Ellipsis maintains various monitoring services for a broad range of customers. Some of these maps are public and can be viewed and used by anyone, others are private and can only be accessed if you have the required login credentials.
