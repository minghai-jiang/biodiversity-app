Het hoofdproduct van Ellipsis is een eenvoudig bruikbare API waarme je onze grote uitgebreide collectie van geo-informatie kunt doorzoeken en visualiseren.

Onze klanten kunnen hun data via onze API opvragen en daarmee eigen analyses doen of eigen applicaties bouwen. Wanneer nodig kunnen we ook volledige service bieden en deze applicaties en alers services voor u maken en hosten binnen het Ellipsis framework.

API

Ellipsis beidt een publieke REST API die directe toegang biedt tot onze geaggregeerde data en een groot aantal aantrekkelijke visualisaties.

Over de data

Ellipsis data wordt ingewonnen en verwerkt op dezelfde resolutie als de orginele databron. We ondersteunen op dit moment 3 typen informatie:

  • Classificatie, om het landbedekking type over de tijd te analyseren.
  • Metingen, om de toestand van een gebied te bepalen.
  • Verandering, om na te gaan of en waar abrupte veranderingen in een landschap zijn opgetreden.

Bovenop de automatisch ingewonnen informatie slaan we gegeorefereerde berichten van gebruikers op via GeoMessage.

We aggregeren deze informatie in zowel standard weMercator tegels als voor de casus specifieke polygonen die per kaart door een opdrachtgever kunnen worden aangedragen.

Toegang

Ellipsis onderhoudt monitoring services voor een groot aantal klanten. Enkele van deze services zijn publiek en kunnen door iedereen bekeken en gebruikt worden, andere kunnen allen bereikt worden met de vereiste login gegevens.
